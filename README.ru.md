# 🌸 Telegram Бот для Цветочного Магазина

[🇬🇧 Read in English](README.md)

Telegram-бот для цветочного магазина с полным циклом покупки: просмотр каталога, управление корзиной, оформление заказа с выбором даты доставки и симулированной или реальной оплатой через Telegram Stars. Включает полноценную админ-панель для управления товарами и заказами.

Портфолио-проект, демонстрирующий реальную архитектуру бота на aiogram 3 с асинхронной ORM SQLAlchemy и FSM-сценариями.

## Возможности

### Для покупателя
- 🌸 Просмотр каталога по категориям с поддержкой фото и пагинацией
- 🛒 Управление корзиной — добавление, удаление, изменение количества
- 🎟 Поддержка промокодов (процентная и фиксированная скидка)
- 📅 Выбор даты доставки из ближайших 5 дней
- 📦 Оформление заказа с выбором типа получения, адреса, телефона и даты
- ✅ Экран подтверждения перед отправкой заказа
- 💳 Оплата через Telegram Stars (реальная) или демо-кнопка (переключается)
- 📋 История заказов с отслеживанием статуса
- ❓ Вопросы по конкретному заказу напрямую через бота

### Для администратора
- 📊 Админ-панель через кнопки меню (без команд)
- 📋 Просмотр всех заказов с полными данными и пагинацией
- ➕ Добавление товаров через FSM-wizard (название, описание, цена, категория, фото)
- ✏️ Редактирование любого поля товара по отдельности
- 🗑 Удаление товаров из каталога
- 🔔 Мгновенные уведомления о новых оплаченных заказах с данными клиента
- 🔧 Смена статуса заказа с автоматическим уведомлением покупателя

## Запуск

### Требования

- Python 3.11+
- Токен бота от [@BotFather](https://t.me/BotFather)

### Установка

1. Клонируй репозиторий:
```bash
git clone https://github.com/Tihonov-Michael/tg-catalog-store-bot.git
cd tg-catalog-store-bot
```

2. Создай и активируй виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # Linux / macOS
```

3. Установи зависимости:
```bash
pip install -r requirements.txt
```

4. Создай файл `.env`:
```
BOT_TOKEN=your_token_here
ADMIN_ID=твой_telegram_id
USE_REAL_PAYMENT=false
```

5. Заполни базу демо-данными:
```bash
python seed.py
```

6. Запусти бота:
```bash
python main.py
```

## Режимы оплаты

Управляется переменной `USE_REAL_PAYMENT` в `.env`:

| Значение | Поведение |
|----------|----------|
| `false` | Демо-кнопка — имитирует оплату мгновенно, без реальной транзакции |
| `true` | Настоящий инвойс Telegram Stars через `send_invoice` |

## Безопасность

- Все запросы к БД через SQLAlchemy ORM с параметризованными выражениями — защита от SQL-инъекций
- Rate limiting через middleware — защита от спама
- Данные пользователей строго изолированы по `user_id`
- Все административные действия проверяют `ADMIN_ID` в каждом хэндлере
- Токен бота и ID администратора хранятся в `.env`, исключены из системы контроля версий
